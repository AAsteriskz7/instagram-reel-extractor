# Privacy Policy for Instagram Reel Insight Extractor

This extension is built with user privacy as a core priority. 

### 1. Data Collection and Usage
- **Local Storage:** Your Gemini API Key and your saved summaries are stored locally inside your browser's extension storage (`chrome.storage.local`).
- **Direct Connection:** When summarizing a post, the extension communicates directly with Google's official Gemini API. 
- **No Third-Party Servers:** There are no database servers, middleman APIs, or third-party servers involved. The developer of this extension does not collect, see, or store your API keys, Instagram credentials, or processed summaries.

### 2. Third-Party Services
This extension interacts with:
- **Instagram:** To retrieve public page content, videos, and images you explicitly choose to summarize.
- **Google Gemini API:** To analyze the media and captions. Please refer to Google's Privacy Policy for how they handle API requests.

### 3. Changes to This Policy
If the extension is updated in the future to change how data is stored, this page will be updated.
